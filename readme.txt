Data Mining subject at FIB-UPC (conducted in English)

In this subject, we completed two group projects. Each project involved selecting, preprocessing, and visualizing data. Following this, we applied various techniques to extract insights based on the project's objectives.

First Project: Clustering Analysis of Speed Dating Data
The goal of this project was to identify patterns within the dataset using clustering techniques.
We used R, Rstudio and Jupyter Notebook/Google Colab for this analysis.

Second project: predict a bank's credit score
The goal of this project was to predict a bank's credit score by training some models with the data. 
We used Python and Jupyter Notebook/Google Colab for this analysis.

For more information, read the Final report of the projects.
